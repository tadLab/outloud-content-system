## Feature 1: Approved Column

### Problem
When Ondrej clicks "Approve", the post currently goes to "Scheduled" but it's not actually scheduled yet. This is confusing.

### Solution
Add new column "Approved" between Final Review and Scheduled.


| Column | Who Acts | What Happens |

| **Approved** | Tade | Ready to schedule â€“ Tade picks date/time |

### Theme Management (in Settings)

See Feature 4: Settings below.

### Database Schema

```sql
-- Themes table
CREATE TABLE themes (
  id UUID DEFAULT uuid_generate_v4() PRIMARY KEY,
  name TEXT NOT NULL UNIQUE,
  color TEXT NOT NULL, -- Hex color
  description TEXT,
  sort_order INTEGER DEFAULT 0,
  is_default BOOLEAN DEFAULT FALSE, -- System defaults can't be deleted
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Add theme to posts
ALTER TABLE posts ADD COLUMN theme_id UUID REFERENCES themes(id);

-- Seed default themes
INSERT INTO themes (name, color, description, is_default, sort_order) VALUES
  ('Case Study', '#3B82F6', 'Project showcases, client work', true, 1),
  ('Behind the Scenes', '#8B5CF6', 'Process, messy middle, how we work', true, 2),
```

### Logic

```typescript
function canSubmitForReview(post: Post): { canSubmit: boolean; reason?: string } {
  // Must have title
  if (!post.title || post.title.trim().length === 0) {
    return { canSubmit: false, reason: 'Title is required' };
  }
  
  // Must have content
  if (!post.content || post.content.trim().length === 0) {
    return { canSubmit: false, reason: 'Content is required' };
  }
  
  // Content must be at least 50 characters
  if (post.content.trim().length < 50) {
    return { canSubmit: false, reason: 'Content must be at least 50 characters' };
  }
  
  // Must have platform and account selected
  if (!post.account_id) {
    return { canSubmit: false, reason: 'Please select an account' };
  }
  
  // AI score must be below 60% (not too AI-ish)
  if (post.ai_score && post.ai_score >= 60) {
    return { canSubmit: false, reason: 'AI score too high. Please rewrite to sound more human.' };
  }
  
  // ToV score should be above 70%
  if (post.tov_score && post.tov_score < 70) {
    return { canSubmit: false, reason: 'Tone of Voice score too low. Please adjust to match brand.' };
  }
  
  return { canSubmit: true };
}
```
